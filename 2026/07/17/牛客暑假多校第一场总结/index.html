牛客暑假多校第一场总结

A题

题意：有一个小写字符串s,当长度恰好为8且第1，3，5，7个字符是辅音字母（即不是’a’，’e’，’i’，’o’，’u’)，第2，4，6，8个字符是元音字母（即’a’，’e’，’i’，’o’，’u’）时输出“Suspected Virus”，否则输出”Well-Being”。

主要算法：模拟

解题思路：首先判断s长度是否为8，如果不是就直接输出，然后分别检查奇数位和偶数位的字母是不是元音，最后再输出结果

E题

题意：对于一个从0到n-1的排列P，定义其权值为$f\left(P\right) = \sum\limits_{0≤i<j<n}(P_j - P_i)$ ，现在给定一个P，求出P的权值。

主要算法：数学公式

解题思路：先分析样例，可以将$(P_1 - P_0) + (P_2 - P_0) + (P_3 - P_0)$拆开得到$P_1 + P_2 + P_3 - 3P_0$，而第二个（即从$P_2 - P_1$开始）则可以拆分成$P_2 + P_3 - 2P_1$，很容易发现，其权值的公式是

$P_1 + … + P_{n-1}$ $-(n-1)P_0$
$P_2 + … + P_{n-1}$ $-(n-2)P_1$
$P_3 + … + P_{n-1}$ $-(n-3)P_3$
$…$ $…$
$P_{n-1}$ $-P_{n-2}$

进一步计算发现公式$(3-n)P_1 + (5-n)P_2 + … + (n-3)P_{n-2} + (n-1)(P_{n-1} - P_0)$，由此得出最终公式为$\sum\limits_{i = 0}^{n-1}(2i + 1 - n) * P_i$
